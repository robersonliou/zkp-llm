# zkp-llm — ZKP PoC demos
Two minimal, reproducible zero-knowledge-proof demos.
- `poc/ezkl-embedding-demo/` — EZKL (Halo2) verifiable embedding step (zkRAG / zkLLM minimal unit).
- `poc/risc0-cartesi-step-demo/` — RISC0 zkVM x Cartesi v0.20 machine step proof.
Each demo is self-contained and runs in Docker; see the per-demo `README.md` for instructions.
